함수
    익명함수 - 변수에 함수를 선언(함수명 없음)
	변수명=function(){};
    화살표함수
	변수명 = ([함수에 쓰일 변수]) => {}
    선언함수 - 선언함수 위치에 관계없이 함수 사용 가능
	function 함수명(){}
    디폴트값을 갖는 매개변수 - 함수내의 변수에 값을 주어 변수 없이 함수 호출 시 함수 내 변수값 사용
	function 함수명(변수1 = a...){}
    가변인자함수-매개변수 갯수에 따라 변하는 함수, 화살표 함수는 x
	ex)Array()함수
    내부함수-덮어쓰기 방지를 위해 함수내 함수를 선언

		function 덮어쓰기방지될 함수명(){} }
    콜백함수-매개변수에 넣어 전달된 함수
	function 함수명(callback){
		....
		callback(); }
    자바스크립트 내장함수
	타이머함수 : setTimeout(function, millisecond) ; 일정시간 후 함수를 한번 실행
		    setInterval(function, millisecond) ; 일정시간마다 함수를 반복해서 실행
		    clearTimeout(id) ; setTimeout 중지
		    clearInterval(id); setInterval 중지
