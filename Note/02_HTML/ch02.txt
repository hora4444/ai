태그
      태그 HTML문서를 구성하는 요소, 열린태그, 닫힌태그로 나뉨

      글자관련 태그
	H1~H6 제목, 숫자가 작을수록 사이즈가 큼
	p 본문
	br 개행(enter)
	a 앵커 (하이퍼링크)
	그 외 b, l, sup, del ...

      목록 태그
	ul : 순서가 없는 목록 태그
	ol : 순서가 있는 목록 태그
	li : 목록 요소
 
      표 태그 (table)
	tr : 표 내부의 행 태그
	th : 행 내부의 열, 제목 셀 태그
	td : 행 내부의 열, 일반 셀 태그

      이미지 태그 (img)
	src : 이미지 경로(상대주소, 웹주소)
	alt : 이미지가 없을 때 나오는 글자지정
	width : 이미지 너비
	height : 이미지 높이

      오디오(audio), 비디오(video)  태그
	src
	preload : 재생 전 모두 불러올지 지정
	loop : 반복재생
	controls : 재생도구를 출력할지 지정
	(비디오만) width, height
